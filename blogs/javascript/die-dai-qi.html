    # 迭代器（interator）

    • 可使用户在容器对象（链表或数组）上遍历的对象，使用该接口无需关系对象内部实现细节。
    • 迭代器是一个用来帮助我们对某个数据结构进行遍历的对象。
    • 在JavaScript中，迭代器是一个具体的对象，这个对象需要符合迭代器协议
      • 在JavaScript中的迭代器标准是一个特定的next方法
    • next方法
      • 一个无参函或只有一个参数的函数（来自最新英文mdn文档），返回一个拥有一下两个属性的对象
      • done（是否迭代完成）：如果迭代器可以产生序列中的下一个值，则为false 如果迭代器已经迭代完毕，则为true。此时value是可选的。
      • value：迭代器返回的任何JavaScript的值。done为true可忽略

    # 可迭代对象

    • 当一个对象符合迭代器协议时，就是一个可迭代对象

    • 这个对象的要求是必须实现 @@interator方法，在代码中使用Symbol.interator 访问该属性

    • //可迭代对象
consr interatorObj = {
    name: ["abc", "cba", "nba"],
    [Symbol.interator]: function() {		//[Symbol.interator]可计算属性

                }else{
                    return {done: false, value: undefined};
                }
            }
        }
    }
}

const interator = interatorObj[Symbol.interator]();	//每次调用都生成一个新的迭代器

    • 迭代器对象 ！== 可迭代对象

    • 迭代器与可迭代对象不是同一个概念，迭代器只需满足迭代器协议；迭代器对象要求对象内部具有[Symbol.interator]属性，这个属性的值是一个函数，调用这个函数返回一个迭代器对象

    # 生成器

    ​ 生成器是ES6中新增的一种函数控制、使用的方案，它可以让我们更加灵活的控制函数什么时候继续执行、暂停执行等。

    ​ 生成器本质上是特殊的迭代器。

    • 生成器函数也是一个函数，但和普通的函数有区别

      • 首先，生成器函数需要在函数名前加*

      • 其次，生成器函数可以通过yield控制函数的停止

      • 调用生成器函数返回一个生成器对象（Generator）,生成器对象是特殊的迭代器对象

      • function* foo(){
	console.log(1);
    yield			//第一段代码结束
    console.log(1);
    yield			//第二段代码结束
    console.log(1);
    yield			//第三段代码结束
    console.log(1);
}
const generator = foo();
generator.next();	//执行第一段代码
generator.next();	//执行第二段代码
generator.next();	//执行第三段代码

    /*
	需求
	1.url: why->why
	2.url: why->whyaaa
	3.url: whyaaa->whyaaabbbccc
*/
function requestData(url) {
    return new Promise((resolve, rejected) => {
        setTimeout(() => {
            resolve(url)
        }, 2000)
    })
}

function* getData() {
    const res1 = yield requestData('why')
    const res2 = yield requestData(res1 + 'aaa')
    const res3 = yield requestData(res2 + 'bbb')
    const res4 = yield requestData(res3 + 'ccc')
    console.log(res4)
}

function execGenerator(genFn) {
    const generator = getData();
    function exec(res) {
        const result = generator.next(res);
        if (result.done) {			//迭代结束
            return result.value
        }
        result.value.then(res => {
            exec(res);
        })
    }
    exec()
}
const finally = execGenerator(getData);		//自动递归迭代
